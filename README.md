# linear-optimization-solvers

A python application with GUI to help students with linear optimization problems.

## Features

| Method              | Implemented |
| ------------------- | ----------- |
| Simplex method      | ✅          |
| Graphical method    | ❌          |
| Dual Simplex method | ❌          |
| Big M method        | ❌          |
| Two-phase method    | ❌          |
| Elipsoid method     | ❌          |
